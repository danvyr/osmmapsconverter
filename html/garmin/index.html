Ежедневные обновления OpenStreetMap карт Беларуси для Garmin

Карты генерируются с помощью mkgmap из данных OSM. Доступно несколько различных стилей разной степени (не)удобства.

Установка карт

Устройство Garmin (навигатор, часы): Скопировать img файл в папку Garmin на устройстве.
Basecamp: Распаковать архив .gmap.tgz и скопировать папку (напр. "Belarus OpenStreetMap.by generic.gmap") в папку с картами BaseCamp (напр. C:\Users\user\AppData\Roaming\Garmin\Maps).

Вопросы и предложения в Telegram:danvyr, на почту danvyr on gmail.com, либо в Issues на странице проекта в GitHub
GitHub: https://github.com/danvyr/osmmapsconverter

Варианты карт

Файлы с _ru в имени — русский язык названий объектов вместо того, что по умолчанию (для Беларуси — белорусский)

generic generic_new rogal stranger
"Дефолтный" стиль mkgmap, без каких-либо кастомизаций. Стиль общего назначения для garmin.openstreetmap.nl, основан на дефолтном mkgmap Стиль польского проекта OSMapa.pl (ныне не существует) Стиль авторства Максима Васильева из клуба "Велосотня", Кирово-Чепецк. От оригинального проекта остался только репозиторий: https://github.com/MaksVasilev/stranger-garmin. Стиль адаптирован для использования в велосипедных внедорожных поездках, добавлены важные для туристов объекты.
Для openstreetmap.by переработаны некоторые иконки POI для однообразия, устранено несколько ошибок.
